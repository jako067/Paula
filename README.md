# Paula App ❤️

Aplicación romántica dedicada a Paula. Sidebar izquierda desplegable con Inicio y 5 razones.

## Uso local

1. Abre `index.html` en el navegador.

## GitHub Pages

1. Push a GitHub repo.
2. Settings > Pages > Deploy from branch `main` (root `/`).
3. Visita `https://tuuser.github.io/repo-name`.

## Personalizar

- Reemplaza `src` en `img src` con tus imágenes URLs.
- Edita textos en `index.html` secciones `razonX`.
- Añade más `<li><button data-page="razon6">Emoji Razón 6</button></li>` + sección `<section id="razon6" class="page">...</section>`.

¡Escalable y lista! 💕
